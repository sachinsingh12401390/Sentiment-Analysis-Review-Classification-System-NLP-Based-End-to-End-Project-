# Sentiment Analysis Web App

## 🚀 Overview
This project is a Sentiment Analysis system that classifies text as **positive or negative** using Machine Learning and NLP techniques.


## 🧠 Features
- Text preprocessing using NLTK  
- TF-IDF vectorization  
- Logistic Regression model (~89% accuracy)  
- Flask API for predictions  
- Interactive web interface  


## 🛠️ Tech Stack
- 1. Python      
- 2. Scikit-learn  
- 3. NLTK  
- 4. Flask  


## 📊 Dataset
IMDb Movie Reviews Dataset (50,000 samples)

## ▶️ How to Run

1. Install dependencies

bash
pip install -r requirements.txt

2. Run the app

bash
python App/app.py

3. Open in browser

http://[IP_ADDRESS]/ui
